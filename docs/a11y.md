# Accessibility (a11y) Best Practices for Hugo & Congo Theme

The Congo theme is designed to be highly accessible out-of-the-box, featuring "skip to content" links, ARIA descriptions, and good keyboard navigation. To maintain and improve this:

## 1. Configuration
- **Language Code**: Ensure `languageCode` is accurately set in your config so screen readers use the correct pronunciation rules.
- **Menus**: Consider adding `title` attributes on menu links if the link text alone isn't perfectly descriptive.

## 2. Content & Media
- **Images**: Always use descriptive `alt` text for images in markdown (`![alt text](image.jpg)`) and within shortcodes.
- **Headings**: Maintain a logical heading structure (H1 -> H2 -> H3) without skipping levels.

## 3. Customizations
- **Layout Overrides**: If overriding layouts in `layouts/`, ensure you don't inadvertently remove ARIA labels or break keyboard navigation focus states.
- **Contrast**: If changing theme colors, ensure high contrast ratios between text and background.
